# Todo Application - Demo App

**Try the app here:** [https://todo-list-shauryas.netlify.app/](https://todo-list-shauryas.netlify.app/)

## Objective

Create a simple Todo application using React and Redux. The application allows users to add, view, update, and delete todo items. Additionally, it displays a list of employees fetched via an API.

## Features

- **Todo List:**
  - Add a new todo item.
  - Display a list of todo items.
  - Delete a todo item.
  - Update a todo item.
  
- **Employee List:**
  - Fetch and display a list of employees using the [DummyJSON API](https://dummyjson.com/users).

- **Responsive Design:**
  - Fully responsive layout, optimized for both mobile and desktop views.

## Technologies Used

1. **React:**
   - Functional components and hooks.

2. **React Router:**
   - Two routes:
     - Todo List
     - Employee List

3. **Redux & Redux Thunk:**
   - Used for state management of todo items.
   - Redux Thunk used for async API calls.

4. **API Integration:**
   - Employee list fetched using [https://dummyjson.com/users](https://dummyjson.com/users).
   - Todo list management is handled locally.

## Setup & Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/todo-application.git
   ```

2. Install dependencies:
   ```bash
   cd todo-application
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

   Visit `http://localhost:3000` to view the app.

## Conclusion

This Todo application demonstrates the use of React, Redux, and API integration for managing a list of todos and employees. The app is responsive and built with best practices in mind.
